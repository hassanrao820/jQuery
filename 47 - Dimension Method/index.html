Dimension Method

We all know margin, padding & border etc. of an elements, but some jQuery built-in method also calculate dimension of an element on a web page.This method set or get the width or height of an element and also cover the concept of excluding and/or including padding, border, and margin.

some of following method to get or set the width.

  1. width():Excludes padding, border and margin
  2. height():Excludes padding, border and margin
  3. innerWidth():Includes the padding
  4. innerHeight():Includes the padding
  5. outerWidth():It includes the padding and border
  6. outerHeight()It includes the padding and border
